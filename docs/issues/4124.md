- **URL:** https://github.com/WebAssembly/binaryen/issues/4124
- **Author:** MaxGraey
- **Created:** 2021-09-05
- **Updated:** 2021-09-05

## Description

Recent [Once Reduction pass](https://github.com/WebAssembly/binaryen/pull/4111) properly handled following CFG in pseudocode:
```cpp
bool initialized = false;
void boo() {
  if (initialized) return;
  ...
  initialized = true;
}
```
but unsound for this:
```cpp
bool initialized = false;
void boo() {
  if (!initialized) {
    ...
    initialized = true;
  }
}
```
