# #6615: Support custom section annotations

- **URL:** https://github.com/WebAssembly/binaryen/issues/6615
- **Author:** tlively
- **Created:** 2024-05-20
- **Updated:** 2024-05-20

## Description

The annotations proposal adds [support](https://github.com/WebAssembly/annotations/blob/main/proposals/annotations/Overview.md#details) for text annotations that encode the contents and placement of arbitrary custom sections. The new wat parser supports annotations in general, so we should implement parsing of the custom section annotations and similarly print the annotations in the printer.
